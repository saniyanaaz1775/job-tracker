# Job Notification — Design System

Purpose
- Foundational design system for the Job Notification App. Intentional, calm, coherent, confident. Built for a premium B2C product.

What this repo contains
- Tokens (color, type, spacing, motion)
- CSS foundations (variables, base typography, layout)
- Component guidelines and example markup (buttons, inputs, cards, layout)

Design constraints (apply everywhere)
- Calm, intentional, coherent, confident — no flashy or playful treatments
- No gradients, no glassmorphism, no neon colors, no noisy animations
- Max spacing scale: 8px, 16px, 24px, 40px, 64px
- Single border-radius token used project-wide
- Interaction transitions: 150–200ms, ease-in-out
- Max text block width: 720px
- Headings: Serif. Body: Sans-serif (16–18px). Line-height 1.6–1.8
- Colors: Background (#F7F6F3), Primary text (#111111), Accent (#8B0000), Semantic success/warning (muted) — usage guidance in tokens

How to use
- Import `styles.css` in pages/components for baseline variables and component styles.
- Use tokens.json for automated tooling or token syncing.
- Follow component rules in `components.md` for consistent markup and spacing.
